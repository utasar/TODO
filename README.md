# TODO Application

The `TODO` repository contains a **Task Management Application** built using **JavaScript**, **CSS**, and **HTML**. This application allows users to manage their day-to-day tasks efficiently by adding, editing, and removing tasks in a simple and intuitive interface.

## Language Composition

- **JavaScript (43.1%)**: Implements the app's functionality, such as adding, updating, and deleting tasks.
- **CSS (33.3%)**: Provides a clean and responsive design for an optimal user experience.
- **HTML (23.6%)**: Structures the application and forms the foundation of the interface.

## Features

- Add tasks to the list dynamically.
- Mark tasks as complete or delete them when no longer needed.
- A responsive design that works seamlessly on different devices.

## How to Use

1. Clone the repository:
   ```bash
   git clone https://github.com/utasar/TODO.git
   cd TODO
   ```

2. Open the application:
   - Open the `index.html` file in your browser.

3. Start managing your tasks:
   - Use the interface to add, edit, mark complete, or delete tasks as needed.
